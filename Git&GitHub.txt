git
 : 형상 관리 도구 중 하나로 분산 버전 관리 시스템!

깃허브 사이트에서 로그인 후
로그인, 마이페이지, 리포지터리에서 새로 생성, 창 닫지 말고
git 검색 후 다운로드, 설치 완료되면 프로그램 검색에 git bash 존재

원하는 폴더에서 우클릭 > git bash 실행
git config --global user.name "닉네임" //닉네임 연결
git config --global user.name "이메일" // 이메일 연결
git config --global user.name  // 닉네임 확인
git config --global user.email // 이메일 확인

git init //해당 폴더에 .git 폴더 생성됨 << 깃허브에 올릴 폴더
git status // 상태 확인, 초기엔 No commits yet 이 나와야 정상
git add .                 //파일 추가 .은 모든 사항은 .git에 추가한다는 뜻
git commit -m "깃허브 체험하기" //
git status  //상태 한번 더 확인
git remote add origin https://github.com/cw01483-ly/testRepository.git
 >> 깃배쉬에서 복사 붙여넣는건 쉬프트인서프

git push origin master //해당 리포지터리에 넣기
